# 🍟 Chips Gestion Pro

Application de gestion pour entreprise de production de chips.

## Déploiement

Ce projet est un fichier HTML statique connecté à Supabase.

### Sur Vercel (recommandé)
1. Uploadez ces fichiers sur GitHub
2. Importez le dépôt sur [vercel.com](https://vercel.com)
3. Cliquez **Deploy** — aucune configuration nécessaire

### En local
Ouvrez `index.html` directement dans votre navigateur.
